### Player Controls

Main Menu

```
    Proceed with the selected option
                [ESC]
                
Navigate Left [←]  [→] Navigate Right      

```

Play State
```
          Hit [H]     [S] Stand 
           
      Exit the program after a round
                 [ESC]          
```
